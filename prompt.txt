## 6. Redraw Mechanism

### Prompt

How to display remaining objects after deletion?

### Output

Implemented:

```c
void redrawCanvas()
```

which:

* Clears the canvas
* Loops through all objects
* Draws them again

using:

```c
switch(shapes[i].type)
```
